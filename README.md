The recent outbreak of H5N1 in cattle in the US (2024), with some cases in humans, sparked new conversations on the pandemic potential of H5N1. Although research on H5N1 exists, it has been historically hard to come up with a reliable estimate of the Case Fatality Ratio and Infection Fatality Ratio, because WHO's definition of a confirmed case is quite stringent, hence causing reported cases to likely be only the tip of the iceberg. If only severe cases are captured, then the fatality ratios will look worse than they are in practice.

All data used in the analysis is publicly available on:
* The WHO's website https://www.who.int/publications/m/item/cumulative-number-of-confirmed-human-cases-for-avian-influenza-a(h5n1)-reported-to-who--2003-2025--20-january-2025
* The John Hopkins University's COVID-19 GitHub repo https://github.com/CSSEGISandData/COVID-19/tree/master/csse_covid_19_data/csse_covid_19_time_series
* The World Bank database (for population data and World Devlopement Indicators) https://data360.worldbank.org/en/search

The clustering approach is described in detail in a separate preprint (Afuleni & Gada et al., 2025) available at https://www.medrxiv.org/content/10.1101/2025.11.08.25339808v1
